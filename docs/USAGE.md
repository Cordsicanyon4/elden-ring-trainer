# 🚀 Usage Guide

### Quick Start
1. Clone the repository
2. Install dependencies
3. Edit config.json
4. Run the trainer

```bash
pip install -r requirements.txt
python -m src.main --config config.json
```

### Advanced Usage
- Modify `src/main.py` to add more features
- Use `src/utils.py` for utility functions
- Custom scripts can be run via `config.json`
